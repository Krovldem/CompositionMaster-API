# CompositionMaster API

API для управления составами изделий (спецификациями).

## Технологии
- ASP.NET Core 8
- Entity Framework Core
- PostgreSQL
- Swagger/OpenAPI

## Запуск
1. Установите PostgreSQL
2. Создайте базу данных
3. Настройте .env файл
4. Запустите dotnet run

## API Endpoints
- /api/composition - управление номенклатурой и спецификациями
- /api/dictionaries - справочники
- /api/reports - генерация отчетов
- /api/search - поиск
